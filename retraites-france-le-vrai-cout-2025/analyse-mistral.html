Ce diagramme représente les flux financiers officiels du système de retraite français. Architecture : Nœud "Budget État" comme transit central (comme dans le modèle ZeeWeb).

Légende des flux :

  • Cotisations Privées (253,1 Md€) : CNAV (155 Md€) + AGIRC-ARRCO (98,1 Md€) + Régimes Spéciaux (3 Md€) + MSA/PL (31 Md€).
  • CSG/Impôts Affectés (25 Md€) : Financement de la Sécu Vieillesse (COR 2025).
  • Recettes Fiscales État (319 Md€) : Budget général (DGFiP 2025).
  • Dette/Emprunt (124 Md€) : Déficit 2025 (DGFiP).
  • Budget État (443 Md€) : 319 Md€ (recettes) + 124 Md€ (dette). ~110 Md€ financent les retraites (65 + 6 + 31 + 8).
  • Total Pensions (407 Md€) : COR 2025.
⚠️ Notes méthodologiques :
  • Les données pensions/cotisations sont en 2024 (COR 2025).
  • Les données budget/dette sont en 2025 (DGFiP).
  • Le nœud Budget État (443 Md€) est un nœud de transit central (comme dans ZeeWeb).
  • Le lien Budget État → Dette (124 Md€) représente le déficit total, dont une partie finance les retraites.
  • ~31 Md€ du déficit sont estimés comme liés aux retraites (estimation ZeeWeb).
